Criterios usados

Cómo interpreta el método

  1. Convierte el modelo a igualdad con holguras, excesos y artificiales cuando corresponde.
  2. En maximización usa la fila Z con criterio de columna más negativa.
  3. La fila saliente se elige con el cociente positivo mínimo.
  4. Si Fase I termina con artificiales positivas, el problema es infactible.
  5. Si no existe fila saliente para una columna entrante, el problema es no acotado.
  6. Los campos numéricos permiten valores como 4, -2, 0.5 o 3.75.

Ejemplos para usar

Modelos rápidos de prueba

Maximización con enteros

Minimización con ≥
